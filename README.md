etc
========
etc是一个快速开发java web的框架。主要用于学习新技术。
技术选型：

管理：
maven依赖和项目管理
git版本控制

后端
Ioc容器spring
web框架springmvc
orm框架mybatis
安全框架shiro
验证框架hibernate validator
任务调度框架quartz
缓存ehcache
数据源druid
日志slf4j+log4j
servlet
jsp模板视图

前台
jquery框架
bootstrap界面
jquery validator验证框架
bootstrap-datepicker日历选择
DataTable表格控件
metisMenu菜单控件
